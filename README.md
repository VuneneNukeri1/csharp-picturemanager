# Picture Manager

A Windows Forms application built in C# that allows users to:

- Import pictures
- Display image previews
- Navigate through images
- Delete images
- Manage pictures stored in a local folder

## Technologies
- C#
- .NET Windows Forms
- Visual Studio 2022
